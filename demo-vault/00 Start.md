Welcome to the [File Explorer Reload](https://github.com/mnaoumov/obsidian-file-explorer-reload/) demo vault. This plugin adds a **Reload File Explorer** command (and **Reload Folder** / **Reload Folder with Subfolders** folder right-click menu items) that refresh the **File Explorer** pane so it matches what is actually on disk - handy after a bulk copy / move / delete done outside Obsidian while it was open.

**How to try it:** run **Reload File Explorer** from the Command Palette, or right-click a folder in the **File Explorer** and choose **Reload Folder** or **Reload Folder with Subfolders**. The pane rebuilds its file list from the file system.

> [!TIP] Interactive buttons
>
> The two setup notes have **Run** buttons, powered by [`CodeScript Toolkit`](https://github.com/mnaoumov/obsidian-codescript-toolkit/), which this vault installs for you automatically on first open (see [[03 CodeScript Toolkit prerequisite]]). Reloading the file explorer is a manual command / menu action, so the feature note has no buttons - just steps to follow.
